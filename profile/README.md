# 紫微知道 · Ziwei Knows

面向紫微斗数、AI 引导式反思与东方人格探索的开源产品。

> Open-source products for Zi Wei Dou Shu, AI-guided reflection, and Eastern archetype exploration.

## 产品生态

| 产品 | 解决什么问题 | 体验与源码 |
| --- | --- | --- |
| **Ziwei Chart · 星盘** | 精准排盘、真太阳时、命盘可视化、流年、合盘与人生趋势。 | [在线体验](https://zwknows.vercel.app/) · [源码](https://github.com/ziweiknows/ziwei-chart) |
| **Ziwei Chat · 问盘** | 基于确定性命盘事实，围绕现实问题进行可追溯的 AI 对话。 | [在线体验](https://ziweichat.vercel.app/) · [源码](https://github.com/ziweiknows/ziwei-chat) |
| **ZATI · 紫曜型格** | 通过行为选择探索十六型东方人格原型。 | [在线体验](https://zati-kappa.vercel.app/) · [源码](https://github.com/ziweiknows/zati) |

三款产品平级互补。请按你当下希望探索自己的方式选择产品；它们共享产品家族，不共享账号、资料或个人数据。
